## Running the project

- Clone this project

```
git clone < project-url.git >
```

- [Install NodeJS](https://nodejs.org/en/) on your computer.

- Install the project on your computer

```
npm install
```

- Then, run the project in executing this in your project folder:

```
npm start
```
